# 会社プロモーションビデオ企画書

対象媒体: Instagram / YouTube / Pinterest
作成日: 2026-07-22

---

## 1. 基本方針:「1ソース・マルチユース」

媒体ごとに動画をゼロから作るのではなく、**60〜90秒のマスター映像を1本制作**し、
そこから各媒体向けに編集して切り出す。撮影は1回で済み、ブランドの世界観も統一できる。

```
マスター映像(60〜90秒・高解像度で撮影)
 ├─ Instagram リール …… 縦 9:16 / 15〜30秒
 ├─ YouTube 本編 ……… 横 16:9 / 60〜90秒(Shorts はリール版を流用)
 └─ Pinterest ………… 縦 9:16 or 2:3 / 6〜15秒ループ
```

※撮影時は縦横どちらにも切り出せるよう、被写体を中央寄りに配置して撮る。

## 2. コンセプト:「プロセス見せ × ビフォーアフター」

デザイン・ものづくり系の事業で3媒体すべてに強い型。

- **手元のアップ**を多用(スケッチ、素材、道具、PC画面、打ち合わせ)
- **自然光**で撮影。色味は白・木目・ニュートラル系に統一
- ナレーションではなく**作業音(ASMR的)+控えめなBGM+字幕テロップ**
- 最後に**完成形のリビール**(ビフォー→アフターのオチ)を置く

派手なエフェクトや速いカット割りより、「丁寧に作っている空気感」を伝える
落ち着いたトーンの方がブランド価値につながる。

## 3. マスター映像の構成(60秒版)

| 時間 | 内容 | ねらい |
|---|---|---|
| 0–3秒 | 完成品の一番美しいカット | フック(離脱防止) |
| 3–15秒 | 依頼の背景・課題+ビフォーの状態(字幕一言) | ストーリーの導入 |
| 15–45秒 | 制作プロセスのモンタージュ | 会社の強み・信頼感 |
| 45–55秒 | 完成リビール+クライアント視点の一言 | 感情のピーク |
| 55–60秒 | ロゴ+CTA | 行動喚起 |

CTA例: 「制作のご相談はプロフィールのリンクから」

## 4. 媒体別の最適化

### Instagram(リール)
- 縦 9:16 / 15〜30秒
- **冒頭2秒に最強のカット**(完成品チラ見せ)
- 無音再生でも伝わるよう字幕必須
- キャプションに制作の裏話+ハッシュタグ(地域名・業種・#インテリアデザイン 等)
- 投稿頻度目安: 週1〜2本(プロセスの小ネタを量産できる)

### YouTube
- 横 16:9 / 60〜90秒のブランドムービーとして掲載
- タイトル・説明文に検索キーワード(「◯◯(地域) オーダー家具」等)を入れる
- リール版をそのまま Shorts にも投稿
- チャンネルのトップ動画(チャンネル紹介)に設定する

### Pinterest
- 縦 9:16 または 2:3 / 6〜15秒の**ループ動画**
- テキストオーバーレイで内容が一目で分かるように(音声に依存しない)
- ピンは検索経由で**数ヶ月〜数年流入が続く**ため、流行より普遍的な美しさ重視
- 1本のマスターから「素材」「工程」「完成」など複数ピンに分割して投稿

## 5. 制作方法と費用感

| 方法 | 内容 | 費用目安 |
|---|---|---|
| 内製 | iPhone+三脚+自然光で撮影、CapCut / DaVinci Resolve で編集 | ほぼ0円(機材除く) |
| 半外注 | 撮影は自社、編集+媒体別切り出しを外注 | 3〜8万円 |
| 外注 | 企画・撮影・編集一式 | 10〜30万円 |

まずは内製で1本作り、反応を見てから外注でクオリティを上げる進め方が低リスク。

## 6. 撮影チェックリスト

- [ ] 完成品のヒーローカット(複数アングル)
- [ ] ビフォー(着手前)の記録
- [ ] 手元アップ: スケッチ / 素材 / 道具 / PC作業
- [ ] 引きの作業風景・打ち合わせ風景
- [ ] 作業音を別録り(環境音)
- [ ] ロゴデータ(透過PNG)・ブランドカラーの確認
- [ ] BGM は商用利用可のものを選定(Artlist / Epidemic Sound / YouTube オーディオライブラリ)

## 7. 次のステップ

1. 題材にする案件(直近の代表作)を1つ決める
2. 上記チェックリストに沿って撮影(半日〜1日)
3. 60秒マスター → 3媒体版を編集
4. 各媒体に投稿し、2〜4週間の再生数・保存数・プロフィール遷移を計測
5. 反応が良かった型で2本目以降をシリーズ化
